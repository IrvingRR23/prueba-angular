# Prueba Angular

Proyecto inicial creado con Angular CLI para validar la configuración del entorno de desarrollo en Windows.

## Objetivo

Este repositorio sirve como primera prueba de entorno para desarrollo frontend con Angular, Node.js, npm, Git, GitHub y Visual Studio Code.

## Tecnologías usadas

- Angular CLI 22.0.1
- Node.js 24.16.0
- npm 11.13.0
- Git
- Visual Studio Code
